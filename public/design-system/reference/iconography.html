Country flags, payment & social
Use the source library directly — they don't round-trip as clean SVGs.

Use the Figma library in-place

Country flags (195), payment logos (40 — Visa/Mastercard/AMEX/Apple Pay/…) and social brand marks (25) live in 06. Icons. Their source shapes are composed — stripes, masks, emblems — and don't export as clean standalone SVGs. When you need them, pull straight from the component library instead of rebuilding them here.

Rules
Don't redraw — copy from library.
  • Design on a 24 × 24 grid, 2 px padding, 20 × 20 live area.
  • Stroke 1.5–2 px, rounded caps/joins. Library is drawn at ~1.5 px.
  • Inherit currentColor. Never hardcode brand red into an icon SVG.
  • Match icon size to the cap-height of adjacent text (24 px icon for 16 px body).
  • Metric glyphs are domain-specific — only use on data screens & sport dashboards.
