Guidelines
Do
  • Fall back to initials when no photo is available — never default placeholder silhouettes.
  • Use first + last initials, uppercase, tracked to 0.02em.
  • Hash the display name to a fixed color so the same user stays the same color.
  • Pair presence dots with aria-labels (e.g. aria-label="Online").
Don’t
  • Stretch non-square photos — always object-fit: cover.
  • Use brand red for non-featured users — reserve red for the current session or admin.
  • Show presence at xs / sm unless the row is > 32 px tall.
  • Skip the 2 px surface ring on the presence dot — it will disappear into a photo.
